AI Camp

At AI Camp, my team and I worked on building an Air Canvas project that can take in live footage from the user's webcam, detect the user's hand guestures (thumbs up, thumbs down, palm, fist, point), and use that to allow the user to draw on the canvas. We also included features for the user to change their brush to any color of the rainbow based on vertical position, switch the brush to erase, increase/decrease the brush size, and upload and image for the canvas' background. The computer vision AI aspect of this project involved using YoloV5, web scraping, labeling images, and model analysis, and the website the Air Canvas was hosted on was programmed using HTML, CSS, and Flask.
