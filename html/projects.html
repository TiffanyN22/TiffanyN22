Coding Projects

Insight

The purpose of Insight is to promote accountablity and safer driving by sending message alerts to user's contacts if they try existing the app while driving or drive over the speed limit. To create this, we used the Inrix incidents API and street segment API, Expo's accelerometer , Vanage API to send text message updates, and firebase to store contacts. Insight was created using React Native for the frontend and Flask for the backend, and it won 2nd place out of over 300 particants at SCU ACM's 2023 Inrix Hackathon!

MyGov

The app is aimed towards increasing civic engagement by helping users learn about candidates. On a federal scale, it allows users to select their representatives and see what they stand for through the legislation they have sponsored. Locally, it lets users it find nearby polling locations and view candidates in local elections. This apps is coded in Swift UIKit and Storyboard and utilizes the Congress.gov API, Google Civil Engagement API, and Google Maps API to get the necessary data.

Focus Pal

This website is an all-in-one program to help students manage their tasks by including a todo list they can save tasks to, a weekly calendar view to schedule when to complete those tasks, and a notes page linked to each task with a timer. This website was programmed using HTML, CSS, and Javascript in January 2023, and tasks, calendar events, and notes are saved using local storage.

Mafia Moderator

This app acts as a moderator for a game of Mafia by allowing the user to enter the number of players and player names, randomly assigning a role for each player, and facilitating the night and voting phase by keeping track of which players are alive and when either team wins. This app was written in December 2022 using Swift, including SwiftUI, UIKit, and Storyboard.

Snake Game AI

The project is an AI for a modified game of Snake that generates a map with random obstacles and food, uses the breadth-first search algorithm to find a path from the food to the snake's head, moves the snake to the food until it reaches its target length or until there is no path from the snake to the food, and includes an option for the user to visualize bfs and/or path generation. The program was coded in python using pygame in November 2022.

Metaphor Generator and Topic Analysis

This program uses GPT3 and OpenAI's API to generate a synthetic dataset of metaphors, sklearn and pandas dataframe to create and train a model to predict the topic of each metaphor, and includes code to generate similes on a user-inputted topic. The project was coded in python in October 2022.

What Do We Do

The purpose of (WD)^2, or What Do We Do, is to help people make plans around an event by finding parking lots and resturants within a specified radius. This website uses the Inrix API to find nearby parking lots, Yelp API to find restaurants, and Google Maps API to display the locations and addresses. This involves using RESTful APIs, and the code was written in HTML, CSS, and Javascript in November 2022.

Most Likely To

This app randomly selects and displays a prompt from a list of over 50 for a game of "Most Likely To." It was written using Swift with SwiftUI in December 2022.
