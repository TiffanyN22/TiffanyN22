X-Ray Musculoskeletal Condition Classification

This project performs binary classification of x-ray images to detect if abnormal musculoskeletal conditions are present using several Python libraries including Pytorch, OpenCV, PIL, Scikit-Learn, Pandas, Numpy, Matplotlib, and SciPy. We compared several preprocessing techniques (PCA, Normalize, Lapclacian Filter, Unsharp Mask, Bilateral Filter, rotation, etc.) and models (LDA, QDA, CNNs, CNNs with dropout and layer normalization).

Ascend

This website helps freelancers with contract negotiations by creating market analysis visualizations for their industry and location, summarizing the main points in lengthy contracts, and automatically drafting negotiation emails with AI.
I was in charge of developing the majority of the frontend in React.js, creating data visualization with matplotlib by retriving data from the CarreerOneStop API using Flask, storing user data in AWS S3, and integrating the frontend and backend. This website won 3rd place for the main AWS AI track in SCU ACM's 2024 AWS-Inrix Hackathon.

MRI Brain Tumor Detection and 3D Reconstruction

This python program uses numpy, cv2, matplotlib, scipy, and nibabel to take MRI scans, create a 3D reconstruction of both the brain and the labeled tumor, and perform image segmentation to detect and separate the tumor using a U-net CNN model.

Baseball Pitch and Hit Data Visualization

This python program uses matplotlib and pyplot to visualize trends across over 1000 json files and create 3D visualizations of the bat and ball before and after a hit. This project was created as part of the 2024 Women in Sports Data Hackathon.

Escape the Cave of Wonders

Inspired by the Disney movie Aladdin, the goal is to help Aladdin escape by completing a series of 14 puzzles scattered around the cave, which includes navigating fire mazes, piecing together puzzles, solving written riddles, gathering materials and tools, and more! Insight

The purpose of Insight is to promote accountablity and safer driving by sending message to alert the user's contacts if they try existing the app while driving or drive over the speed limit. To create this, we used the Inrix incidents API and Segment Speed API, Expo's accelerometer , Vonage API to send text messages, and Firebase to store contacts. Insight was created using React Native for the frontend and Flask for the backend, and it won 2nd place out of over 300 participants at SCU ACM's 2023 Inrix Hackathon!

MyGov

The app is aimed towards increasing civic engagement by helping users learn about candidates. On a federal scale, it allows users to select their representatives and see what they stand for through the legislation they have sponsored. Mafia Moderator

Snake Game AI

The project is an AI for a modified game of Snake that generates a map with random obstacles and food, uses the breadth-first search algorithm to find a path from the food to the snake's head, moves the snake to the food until it reaches its target length or until there is no path from the snake to the food, and includes an option for the user to visualize bfs and/or path generation. The program was coded in python using pygame in November 2022.

Metaphor Generator and Topic Analysis

What Do We Do

The purpose of (WD)^2, or What Do We Do, is to help people make plans around an event by finding parking lots and resturants within a specified radius. This website uses the Inrix API to find nearby parking lots, Yelp API to find restaurants, and Google Maps API to display the locations and addresses. This involves using RESTful APIs, and the code was written in HTML, CSS, and Javascript in November 2022.
